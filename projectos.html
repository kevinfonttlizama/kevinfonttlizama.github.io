Projecto Interlinked

Interlinked es una aplicacion web fullstack que esta en proceso de desarrollo junto a un equipo de 4 desarrolladores entusiastas del codigo y un Project manager buenas ideas

Esta aplicacion consiste en un modelo de inversiones para projectos de desarrollo de software donde tendra una interfaz comoda para el usuario y se dividira en secciones de emprendedor,desarrollador e inversor

interlinked se desarrolla con HTML, CSS personalizado, Javascript , Typescript & Angular

Quizas en el futuro se añadan nuevas tecnologias para el desarrollo del projecto asi que esta seccion ira recibiendo actualizaciones y tendra mas detalles de su desarrollo con el tiempo hasta finalizar

Data visualizer con D3

Esto es un grafico de visualizacion de datos usando la libreria de la biblioteca de Javascript usando D3 el projecto consiste en un grafico de barras que muestra la evolucion del el indice del PIB de Estados Unidos a travez de una linea de tiempo dividida por años

El Projecto se conecta a una API JSON publica utilizando el metodo Fetch() de Javascript de la cual obtiene los datos, por lo cual usando esos mismos datos el tamaño de las barras se organiza automaticamente de forma ascendente
